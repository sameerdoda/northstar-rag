# 100-Day Director / Principal PM AI Prep Checklist


## Week 1

### Day 1 — Define target role stack
- [ ] Write down your exact target titles: Director of Product and Principal PM
- [ ] List 15 target companies across Canada and the US
- [ ] Choose primary focus: AI platform, AI product, or AI-enabled growth
- [ ] Write 5 bullets on why you are credible for these roles
- [ ] Create a single folder for all job-search artifacts

### Day 2 — Analyze target job descriptions
- [ ] Collect 8 live job descriptions for Director or Principal PM roles
- [ ] Highlight repeated requirements across the job descriptions
- [ ] List the top 10 skills being screened for
- [ ] Mark each skill as strong, medium, or weak for yourself
- [ ] Write a one-page gap summary

### Day 3 — Write career narrative v1
- [ ] Draft your career story from early PM work to HelloFresh
- [ ] Write a 90-second Tell me about yourself answer
- [ ] Write a 30-second executive intro version
- [ ] Practice both versions out loud 3 times
- [ ] Record yourself once and note weak spots

### Day 4 — Build leadership story bank
- [ ] List 10 leadership situations from your career
- [ ] Choose 5 strongest stories for interviews
- [ ] For 2 stories write Situation, Task, Action, Result
- [ ] Add concrete metrics to each story
- [ ] Tag each story by theme: strategy, execution, conflict, failure, influence

### Day 5 — Refresh product sense framework
- [ ] Review one product sense framework you will use consistently
- [ ] Practice Improve Uber Eats in writing
- [ ] Structure the answer as user, pain point, prioritization, solution, metrics
- [ ] Time-box yourself to 30 minutes
- [ ] Rewrite the answer more crisply

### Day 6 — Refresh metrics framework
- [ ] Review north star, input, output, and guardrail metrics
- [ ] Build a metric tree for Spotify or Instacart
- [ ] Write 3 risks of optimizing only one top-level metric
- [ ] Practice explaining metric tradeoffs out loud
- [ ] Save your metric-tree template

### Day 7 — Weekly review and reset
- [ ] Review all outputs from Days 1 to 6
- [ ] List your top 3 weak areas
- [ ] Rewrite one weak answer from scratch
- [ ] Set goals for Week 2
- [ ] Clean and organize your files
